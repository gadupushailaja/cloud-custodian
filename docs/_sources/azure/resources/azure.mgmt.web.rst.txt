.. _azure.appserviceplan.actions.resize-plan:

resize-plan
+++++++++++
Resize App Service Plans

.. code-block:: yaml

  policies:
    - name: azure-resize-plan
      resource: azure.appserviceplan
      actions:
       - type: resize-plan
         size: F1
         count: 1
